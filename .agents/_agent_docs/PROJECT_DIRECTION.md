# Project Direction

## Core Idea
This is a visual editor for scene structure and logic, not a full game engine.

## Main Goals
- visualize objects and their relationships
- understand what each object does
- attach logic descriptions to objects
- edit logic in human-readable text (Russian-friendly)

## What we are NOT building
- no runtime simulation
- no physics
- no ECS
- no execution graph
- no AI generation
- no complex systems

## Logic Philosophy
Logic is currently:
- descriptive
- editable
- human-readable
- not executed

## Priority
Clarity > complexity  
Usability > technical depth
