# Архитектура движка Agenix 2D

Этот файл — "карта" проекта для тебя и будущих AI-агентов. Он помогает быстро понять, как всё устроено.

## 🛠 Технологический стек
- **Framework**: React + Vite + TypeScript.
- **Graphics**: Konva.js (`react-konva`). Используется для 2D-рендеринга с поддержкой слоев.
- **State**: Zustand. Разделен на два стора:
    1. `useCanvasStore`: Данные камеры (X, Y, Zoom) и список всех игровых объектов.
    2. `useUIStore`: Состояние интерфейса (открытые меню, модалки).

## 🗺 Система координат
- **World Space (Мировое пространство)**: Координаты объектов на бесконечной сетке.
- **Screen Space (Экранное пространство)**: Реальные пиксели в браузере.
- *Важно:* Все расчеты в коде преобразуются через формулу: `(ScreenXY - CameraXY) / Scale`.

## 🎨 Стилизация
- **Vanilla CSS**: Темы заданы через CSS переменные в `index.css`.
- **Theme**: Dark Mode с эффектами стекла (Glassmorphism).

## ⚙️ Оптимизации
- **Grid LOD**: Сетка динамически упрощается при отдалении камеры, чтобы не нагружать процессор.
- **Transient Updates**: Изменение координат объектов происходит через Zustand напрямую в Konva, не вызывая тяжелых перерисовок всего React-дерева.
