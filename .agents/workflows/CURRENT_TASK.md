# CURRENT TASK

## Active Phase
Phase 10A — Устранение dangling ссылок логики (Logic Integrity Cleanup)

## Goal
Избежать накопления мертвых ID объектов в массивах `relatedObjectIds` текстовой логики, сохранив при этом сами правила.

**Статус**: ЗАВЕРШЕНО (Completed)

## Что было реализовано
- При удалении объекта (вызов `removeObject` в `useCanvasStore.ts`) теперь автоматически очищаются связанные с ним ссылки в массивах `relatedObjectIds` всех существующих правил (`logicItems`).
- Опционально добавлена кнопка "Delete" в панель Inspector для удобства удаления выбранного объекта в UI.
- При импорте сцены (`sceneValidation.ts` и `App.tsx`) происходит "санитизация" `logicItems`:
  - Извлекаются все идентификаторы объектов, существующих в импортируемой сцене.
  - Из `relatedObjectIds` каждого правила удаляются те ID, которых нет в этом наборе.
  - Сообщается количество удаленных битых ссылок при импорте.
